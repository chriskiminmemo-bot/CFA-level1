# CFA Level I PWA — Deploy Instructions

## Files in this folder
- index.html — Home screen (app hub)
- guide.html — Concept Guide (76 concepts)
- exam.html  — Mock Exams + Test Bank (228 questions)
- manifest.json — PWA manifest
- sw.js — Service worker (offline support)
- icon-192.png — App icon (small)
- icon-512.png — App icon (large)

---

## Option A: Netlify (Easiest — No account needed)

1. Go to https://netlify.com
2. Click "Deploy manually" (drag & drop zone on homepage)
3. Drag the entire CFA_PWA folder into the browser
4. You'll get a URL like: https://amazing-name-123.netlify.app
5. Done! Share this URL.

**On iPhone:**
- Open the URL in Safari
- Tap Share (□↑) → "Add to Home Screen"
- Tap "Add" → App icon appears on your home screen!

---

## Option B: GitHub Pages (Free, permanent URL)

1. Create account at github.com
2. New repository → name it "cfa-level1" → Public
3. Upload all files in this folder
4. Settings → Pages → Source: main branch / root
5. URL: https://[yourusername].github.io/cfa-level1

---

## Option C: Just open locally (no internet needed)

1. Save all files to your phone (iCloud Drive or Files app)
2. Open index.html with Safari
3. Tap Share → "Add to Home Screen"
(Offline mode works — all content is self-contained)
